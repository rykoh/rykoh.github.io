For the last few semesters of my college career I led a small team of a developers in order to continue this project that another student began. The goal was to develop a dashboard for students to explore COVID-19 related research opportunities. The first version was a simple web app built using Streamlit. With scale and more features in mind, we decided to switch it to Flask with some Dash components. Soona fter it was decided that this was not the right framework for the cision we had, and for the final version that is currently being developed, React was a good fit. React will allow this project to be expanded outside of its original intentions.
